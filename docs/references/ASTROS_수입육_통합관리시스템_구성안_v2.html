②계약서 처리반자동 (담당자 확인 필수)완전 자동: 메일 수신 → PDF 파싱 → DB 등록 → 폴더 저장 all-in-one. 이상 시에만 담당자 알림.강화
③선적서류 처리반자동 (담당자 확인)자동 매칭: 수신 → BL 추출 → 계약건 자동매칭 → 선적정보 입력 → 폴더 저장 자동.강화
④메일 초안일부 자동모든 반자동 단계에 메일 초안 자동생성. 담당자는 확인+클릭만.신규
⑤재고 분류단일 재고3단계: 미착재고 / 도착재고 / 상품재고. 각각 다른 평가 기준 적용.대대적 변경
⑥환율 적용없음BL date 기준 서울외국환중개 매매기준율 자동 조회 및 원화 환산.신규
⑦관세 반영없음수입신고필증 PDF 파싱 → 관세 자동 추출 → 제품 원가에 자동 반영.신규
⑧출고 방식출고지시서 생성재고 대시보드에서 원클릭 판매/출고. 수량·단가·판매일 입력 → 출고증 자동생성.강화
⑨창고 FAX이메일만출고증 FAX 자동 발송 (인터넷FAX 서비스 연동). 원클릭.신규
⑩회계 마감없음월별 기초/기말 재고 금액 자동 산출. 3단계 재고별 환율 적용.신규
⑪기말재고 계산없음해당월 판매분 자동 차감하여 기말재고 산출.신규
✅ 설계 원칙 (v2) 1) 자체 완결형: 이카운트 없이 발주→입고→재고→매출→회계마감까지 올인원.
2) 최대 자동화: 사람이 꼭 판단해야 하는 것만 반자동, 나머지는 완전 자동.
3) 반자동 = 메일 초안 포함: 담당자 확인이 필요한 모든 단계에 메일 초안이 자동 생성됨.
4) 회계 연동: 3단계 재고관리 + BL환율 + 관세 반영으로 회계마감 직결 데이터 제공.
5) 장애 안전장치: Excel 자동백업 + 오프라인 운영 매뉴얼 유지.

📊 3단계 재고관리 & 회계 마감

미착재고 / 도착재고 / 상품재고의 정의, 금액 산출 로직, 월별 회계 마감 프로세스입니다.

재고 3단계 정의

🚢 미착재고 (In-Transit)
  • 정의: 선적 완료(BL 발행) ~ 입항 전. 해상 운송 중인 물량.
  • 진입: BL 발행 시 자동 등록
  • 퇴출: 입항 확인 시 → 도착재고로 전환
  • 금액: USD 금액 × BL date 매매기준율
  • 특징: 물리적으로 아직 한국에 없음. 소유권만 이전된 상태.
⚓ 도착재고 (Arrived/미통관)
  • 정의: 입항 완료 ~ 통관 완료 전. 보세 구역에 있는 물량.
  • 진입: 입항 확인 시 (미착→도착 자동전환)
  • 퇴출: 통관 완료 시 → 상품재고로 전환
  • 금액: USD 금액 × BL date 매매기준율 (미착과 동일)
  • 특징: 물리적으로 한국에 있으나 통관 전이라 판매 불가.
✅ 상품재고 (Cleared Stock)
  • 정의: 통관 완료. 창고에 보관 중. 판매 가능한 물량.
  • 진입: 통관 완료 시 (도착→상품 자동전환)
  • 퇴출: 출고(판매) 시 → 판매완료
  • 금액: (USD 금액 × BL환율) + 관세
  • 특징: 관세가 원가에 포함됨. 이 금액 기준으로 마진 계산.

금액 산출 공식

── 미착재고 / 도착재고 원화 금액 ──
원화금액 = USD_계약금액 × BL_date_매매기준율

── 상품재고 원화 금액 (통관 후) ──
상품원가 = (USD_계약금액 × BL_date_매매기준율) + 관세
제품당_원가 = 상품원가 ÷ 총수량(kg)

── 매매기준율 조회 ──
BL_date_매매기준율 = 서울외국환중개 USD/KRW 매매기준율 @ BL_date
※ BL date가 영업일이 아닌 경우, 직전 영업일 환율 적용

── 관세 산출 (수입신고필증에서 추출) ──
관세 = 수입신고필증 상 관세금액 (AI 파싱으로 자동 추출)
제품당_관세 = 관세 ÷ 총수량(kg)

월별 회계 마감 (기초/기말 재고)

══════════ 월별 기초재고 ══════════
기초재고(당월) = 기말재고(전월)

══════════ 월별 기말재고 ══════════

① 미착재고 기말
미착재고_기말 = Σ (해당월 말일 기준 "미착" 상태인 각 건의 USD금액 × BL환율)

② 도착재고 기말
도착재고_기말 = Σ (해당월 말일 기준 "도착" 상태인 각 건의 USD금액 × BL환율)

③ 상품재고 기말
상품재고_기말 = 상품재고_기초
    + 당월_통관완료_입고금액 (원가+관세)
    − 당월_판매금액(원가기준) ★ 해당월 판매분 제외

④ 총 기말재고
총_기말재고 = 미착재고_기말 + 도착재고_기말 + 상품재고_기말

7담당자 확인반자동이사님 또는 담당자가 보고서 확인 후 "마감 승인". 수정 필요 시 조정 가능.

• 상품재고의 판매분 제외는 원가 기준으로 차감 (판매단가가 아닌 취득원가).
• 월 중에 미착→도착→상품으로 전환된 건은 각 상태별 체류 기간에 맞춰 분류됨.
• 첫 달(시스템 도입월)의 기초재고는 수동 입력 필요.

원가 구조 예시

── 예시: 비브라 닭발 35g up, 1 컨테이너 ──

계약금액 = USD 42,000 (28MT × USD 1,500/MT)
BL date = 2026-03-15
BL 매매기준율 = 1,345.50 원/USD

미착/도착 재고 원화:
원화금액 = 42,000 × 1,345.50 = 56,511,000원

통관 후 (상품재고):
관세 = 수입신고필증 상 관세금액: 1,130,220원 (관세율 2%)
상품원가 = 56,511,000 + 1,130,220 = 57,641,220원
제품당 원가 = 57,641,220 ÷ 28,000kg = 2,059원/kg
